Anemia Detection using Machine Learning
📌 Project Overview

This project is a Machine Learning-based system designed to detect Anemia using clinical parameters such as CBC (Complete Blood Count) report values. The model analyzes patient data and predicts whether a person is anemic or non-anemic, helping in early diagnosis and medical support.

🎯 Objective

To build an intelligent system that:

Predicts anemia using blood test parameters
Reduces manual diagnosis time
Assists healthcare professionals in early detection
Provides accurate and fast results using ML models
